## 📌 What is Cryptify?

Cryptify is a security application designed to provide protection to users' digital assets.

It includes features like File Encryption, Folder Locking, Drive Encryption, and Internet Access Control using Password or Face Encryption.

Developed by **Fox Hackerz**.

---

## 🛠 Project Details

Cryptify is built using Python.

---

## ✨ Features

### 🔒 File Lock
Encrypts any file format using multiple encryption techniques.

### 📁 Folder Lock
Converts directories into compressed containers and applies encryption.

### 💽 Drive Lock
Uses Windows BitLocker to encrypt entire partitions.

### 🌐 Lock Internet
Blocks internet access at the operating system level.

### 🧬 Face Encryption
Uses Python’s face-recognition library to:
- Detect and store a face  
- Verify identity during unlocking  

Can be combined with password protection.
